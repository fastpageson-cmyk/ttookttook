## Phase 1 — MVP 구현 (Must = 0단계 + 1주차) — 2026-07-19 완료
화면 번호는 [[화면기획서]] 3-1~3-12 기준. 전부 실제 브라우저 E2E로 검증(콘솔 에러 0).
- [x] 1. 스플래시/온보딩 — 새 슬로건 + 카드 3장 스와이프 + 재방문 시 생략
- [x] 2. 시작 화면(닉네임 입력 — 회원가입 없음)
- [x] 3. 사전 진단 20문항 — 진행률·이전 이동·중도이탈 임시저장(재진입 시 이어서)
- [x] 4. 진단 결과 — 점수 카운트업 + 또래 평균 62.6점 비교 막대 애니메이션
- [x] 5. 0단계 블라인드 자유투자 — 종목 8+지수ETF, 520주, 재생/빨리감기/+4주, 매수·매도 바텀시트(수수료 0.2% 자동 계산), 보유 표시, 자동/수동 종료
- [x] 6. 0단계 종료 리포트 — 수익률 카운트업·MDD·매매횟수·수수료·지수ETF 주간적립(DCA) 비교선 그래프
- [x] 7. AI 매매 실수 리포트 — **규칙 기반** 6종 룰(최대손실 종목/팔고 오른 매매/잦은 매매·수수료/몰빵/벤치마크 비교/무매매)에서 심각도 상위 3건. `generateMistakeReport()` 시그니처 유지로 추후 LLM 교체 가능
- [x] 8. 홈/대시보드 — 다음 액션 카드 + 진도 6단계 + 커리큘럼(2~6주차 잠김)
- [x] 9. 1주차 강의 카드 10개 — 원고 그대로, SVG 인포그래픽 2종 포함, 열람 추적
- [x] 10. 실습 3종 — ①0단계 복기(금리 국면 밴드 클릭 + 내 매매 오버레이) ②금리 인하기 리플레이(체크포인트 질문 2회+균형 해설) ③한미 금리차 슬라이더 위젯(예측→정답 공개)
- [x] 11. 확인 퀴즈 5문항 — 통과 기준 4/5, 오답 시 해당 강의 카드로 점프 링크, 재도전
- [x] 12. 마이페이지 — 진행 이력 + 전체 초기화
- [x] 전체 흐름 E2E: 닉네임 → 진단 → 0단계 → 리포트 → AI 리포트 → 강의 → 실습 3종 → 퀴즈 통과 → 홈 "1주차 완료 🎉"까지 한 사이트 안에서 끊김 없이 동작 확인 (2026-07-19, 콘솔 에러 0, `npm run build` 통과)
