# 프로젝트 기획서 — 뚝뚝

## 1. 프로젝트 개요

| 항목 | 내용 |
|---|---|
| 프로젝트명 | 뚝뚝 (가칭) |
| 한 줄 정의 | 아무것도 모른 채 투자를 해보고 실패를 겪은 뒤, 그 실패를 교재 삼아 거시경제~부동산까지 배우는 6주 투자 교육 앱 |
| 배경 | 20대 금융이해력 62.6점(전 연령 최하위), 매매 회전율은 전 연령 최고. 지식 없이 공격적으로 행동하는 구조적 문제 |
| 기간 | 0단계(체험) + 정식 6주 프로그램 (세션당 20~30분, 주 5회 접속 기준) |
| 프로젝트 성격 | 학교 토이 프로젝트 (2026-07-18 확정) — 유사투자자문업 등록 등 금융 규제 검토 불필요 |
| 플랫폼 | 웹사이트 (2026-07-18 확정) |
| 개발 도구 | Claude Code로 구현 예정. 세부 기술 스택은 Claude Code가 판단 |
| 문서 상태 | 초안 — 하단 "확인이 필요한 사항" 확정 전까지 개발 착수 불가 |

## 2. 목표 및 성공지표(KPI)

**학습 효과 지표**
- 사전·사후 금융이해력 진단 점수 변화(20문항, 한은 조사 문항 기반)
- 0단계 대비 이후 주차 실습에서의 의사결정 변화(예: 1주차 실습에서 금리 구간 인지 후 매매 판단이 달라지는지)
- 3-4주차 "저평가 기업 찾기" 게임, 5주차 "종목선정 vs ETF 비교" 실습에서의 정답률/선택 변화

**서비스 지표**
- 6주 완주율(졸업률) — 유한 코스 설계이므로 완주율이 핵심 지표
- 주차별 이탈 지점(어느 주차에서 가장 많이 이탈하는지). 특히 0단계 → 1주차 전환 구간(AI 리포트가 실제로 동기부여로 이어지는지)을 우선 측정
- 0단계 결과 카드 공유 횟수(확산 가설 검증용)

목표 수치(완주율 등)는 베타 테스트 이후 역산해서 설정하는 것을 제안한다.

## 3. 타겟 사용자

| 구분 | 정의 | 특징 |
|---|---|---|
| Primary | 사회초년생 23~32세 | 첫 월급~시드머니 형성기. 종목토론방·유튜브 의존, 판단 기준 부재 |
| Secondary | 투자 시작 직전 대학 고학년·취준생 | 첫 계좌 개설 전 "예방 접종" 가능한 유일한 시점 |

## 4. 커리큘럼 (개편안 — [[기본적인 컨셉과 개요.]] 3번 참고)

| 단계 | 주제 | 실습 유형 |
|---|---|---|
| 0단계 | 무지식 블라인드 자유투자 | 자유매매형 블라인드 시뮬레이션 (금리 변동기 포함 구간) |
| 1주 | 거시경제와 투자의 이유 | 0단계 복기 + 금리·환율 가이드형 시나리오 3종 |
| 2주 | 저축·절세·정책상품 | 세전/세후 수익 비교 계산기 |
| 3-4주 | 기업분석·가치투자 | 블라인드 재무제표로 저평가 기업 찾기, 장기 vs 단타 비교 |
| 5주 | ETF·인덱스 투자 | 종목선정 vs ETF 적립식 비교 |
| 6주 | 부동산·자산관리 종합 (졸업) | 종합 정리, 나의 투자 원칙 문서화 |

## 5. 기능 요구사항

**Must (MVP — 0단계 + 1주차)**
- 회원가입/로그인
- 사전 진단 20문항 + 또래 평균 비교
- 0단계 블라인드 자유투자 시뮬레이션(금리 변동기 포함 구간, 종목 10개 내외 + 지수 ETF)
- 매수/매도 주문, 수수료·세금 반영 체결
- 0단계 종료 리포트(수익률, 매매횟수, 수수료 총액, MDD, 지수 대비 비교선)
- **AI 매매 실수 분석 리포트** (손실기여 매매 3건 자연어 해설) — 서비스 내 유일한 AI 필수 지점
- 1주차 거시경제 강의 콘텐츠(정적 텍스트·이미지, 사람이 작성)
- 1주차 실습 3종: 0단계 결과-금리구간 복기, 금리 인하기 S&P500 가이드 리플레이, 한미 금리차·환율 실습
- 확인 퀴즈(통과해야 다음 진행)

**Should (2단계 — 2주차, 3-4주차)**
- 2주차 저축·절세·정책상품 강의 + 세전/세후 계산기 실습
- 3-4주차 재무제표·PER·PBR·가치투자 강의 + 저평가 기업 찾기 게임 + 장기 vs 단타 비교 실습

**Could (3단계 — 5주차, 6주차/졸업)**
- 5주차 ETF·인덱스 강의 + 종목선정 vs ETF 비교 실습
- 6주차 부동산·생애주기 자산배분 강의(주택청약 포함)
- 나의 투자 원칙 1페이지 문서 출력
- 졸업 리포트(사전·사후 진단 비교), 실전 전환 체크리스트
- 졸업생 멘토/리뷰어 순환 구조

**제외 확정 (원안에서 축소)**
- AI 뉴스 헤드라인 블라인드 재작성 — 커리큘럼이 고정 강의형으로 바뀌며 실익 감소
- AI 기반 레슨 개인화 — 동일 이유로 제외
- 가계부 작성법·소비 습관 콘텐츠 — 뱅크샐러드·토스 등 기존 서비스와 차별화 어려움
- CAPEX, 애널리스트 리포트 정독법 — 완전 초보 대상에는 과도한 밀도, 심화 옵션으로 이후 검토

**Out of scope (이번 범위 제외, 필요시 재검토)**
- 실제 증권사 계좌 연동
- 실시간 시세 기반 실전 투자 기능
- 특정 증권사·상품 추천(이해상충 방지 원칙상 명시적 배제)
- 레버리지 vs 원칙 투자 비교 실험(원안 4주차) — 이번 개편에서 커리큘럼 밖으로 이동, 필요시 심화 콘텐츠로 재검토

## 6. 비기능 요구사항

| 구분 | 요구사항 |
|---|---|
| 콘텐츠 정확성 | AI 리포트(0단계 매매 실수 분석)는 배포 전 검수 절차 필요. 나머지 6주 분량 강의 콘텐츠는 Claude가 작성하고 팀이 검수 |
| 데이터 무결성 | 0단계 블라인드 시뮬레이션의 종목명·연도 익명화가 검색으로 역추적되지 않아야 함 |
| 접근성 | 세션 20~30분 기준 모바일 우선 설계 필요 |
| 확장성 | 향후 종목 수 확장, 심화 콘텐츠(CAPEX 등) 추가를 고려한 콘텐츠 구조 |
| 법적 검토 | ~~해당 없음~~ — 학교 토이 프로젝트로 확정되어 규제 검토 불필요 (2026-07-18) |
